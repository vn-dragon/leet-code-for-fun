# LeetCode for Fun

A personal collection of LeetCode solutions written in **Go** and **Python**, created just for the fun of it.

## Overview

- **118+ problems** solved and counting
- Every problem is implemented in both **Go** and **Python**
- Solutions live in the `daily/` folder, named by problem number and title

## Topics Covered

| Topic | Problems |

| Binary Search | Search in Rotated Sorted Array I & II, Find Minimum in Rotated Sorted Array, Koko Eating Bananas, Capacity to Ship Packages, Find Smallest Divisor, H-Index II, Find K Closest Elements, Split Array Largest Sum, Minimum Days to Make Bouquets |
| Trees | Inorder/Preorder/Postorder Traversal, Same Tree, Level Order Traversal, Maximum/Minimum Depth, Invert Binary Tree, Balanced Binary Tree, Diameter of Binary Tree, Two Sum IV (BST) |
| Sliding Window | Longest Substring Without Repeating Characters, Minimum Window Substring, Permutation in String, Longest Repeating Character Replacement, Max Consecutive Ones III, Fruit Into Baskets, Subarrays with K Different Integers |
| Two Pointers | Two Sum, Add Two Numbers, Valid Palindrome II, Rotate Array, Sort Array By Parity, Boats to Save People, 3Sum with Multiplicity, Long Pressed Name |
| Monotonic Stack | Next Greater Element II, Sliding Window Maximum, Next Greater Node in Linked List, Shortest Subarray with Sum at Least K |
| Binary Search (Value Space) | Sqrt(x), Sum of Square Numbers, Valid Perfect Square, Kth Smallest in Sorted Matrix, Online Election, Random Pick with Weight |
| Greedy | Bag of Tokens, Most Profit Assigning Work, DI String Match, Partition Labels, Boats to Save People |
| Math & Simulation | Happy Number, Ugly Number III, Find N Unique Integers Sum to Zero, Convert Integer to Sum of Two No-Zero Integers, Flipping an Image |

## Running Solutions

**Go**
```bash
cd daily
go run <problem-file>.go
```

**Python**
```bash
cd daily
python3 <problem-file>.py
```
